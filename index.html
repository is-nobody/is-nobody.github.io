Why Apex?

Simplicity

Clean, indentation-based syntax with no braces or semicolons. Designed to be readable and easy to learn.

Built-in Power

Comes with a comprehensive standard library (os, sys, math, string, table, ffi, random, codecs) out of the box.

Performance

Written in C with a register-based virtual machine, nan-boxing for efficient value representation, and optimized bytecode execution.

Bundle Binaries

Use apex build to package your source with interpreter into a single standalone executable.
